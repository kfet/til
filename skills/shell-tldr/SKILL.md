# Use TLDR pages

TLDR pages - https://tldr.sh

```bash
tldr tar
```

Install the minimal [tldr-sh-client](https://github.com/raylee/tldr-sh-client)
```bash
mkdir bin
curl -o ~/bin/tldr https://raw.githubusercontent.com/raylee/tldr/master/tldr
chmod a+x ~/bin/tldr
```
